# Codex Hub Memory Template

这是 `Codex Hub` 产品化版本的模板记忆库。

用途：

1. 给新部署者一套最小可运行的长期记忆骨架
2. 不包含任何个人真实项目、真实历史和私有状态
3. 为后续 bootstrap 自动初始化提供默认结构

- `PROJECT_REGISTRY.md`
- `ACTIVE_PROJECTS.md`
- `NEXT_ACTIONS.md`
- `01_working/`
- `02_episodic/daily/`
- `03_semantic/projects/`
- `03_semantic/systems/`
- `07_dashboards/`
