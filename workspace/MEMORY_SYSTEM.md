# Codex Hub Product Memory System

这份 `memory/` 目录是 `Codex Hub` 产品化版本的**模板记忆骨架**。

它不是个人真实长期记忆库，而是给部署者一套最小可运行的结构。

## 事实源分层

固定链路保持不变：

`专题板 -> 一级项目板 -> NEXT_ACTIONS -> 07_dashboards`

## 模板目录

- `PROJECT_REGISTRY.md`
- `ACTIVE_PROJECTS.md`
- `NEXT_ACTIONS.md`
- `01_working/`
- `03_semantic/`
- `07_dashboards/`

## 原则

1. 模板库不预置真实项目状态
2. 模板库不预置个人项目和私有记忆
3. 模板库只提供结构，不提供个人历史事实

## 后续工作

后续产品化版本应支持：

1. bootstrap 自动生成模板文件
2. acceptance 自动检查 memory 骨架完整性
3. 用最小站点配置驱动 workspace 和 memory 的绑定关系
